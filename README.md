# Leave Assistant Bot

A simple Python bot that behaves like an AI assistant for leave requests.

## Features
- Answers simple leave-related questions.
- Helps users create a leave form interactively.
- Lets users change the greeting at runtime.

## Run
```bash
python bot.py
```

## Example commands
- `help`
- `what is leave form`
- `create leave form`
- `change greeting to Hello team, I can help with leave requests!`
- `show greeting`
- `exit`

## Test
```bash
pytest -q
```
